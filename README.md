# Big-Data-Lab
# Exercise 2 - HDFS Basic Commands

## Objective

To learn and perform basic Hadoop Distributed File System (HDFS) commands for file and directory management.

## Prerequisites

- Ubuntu 22.04 (WSL)
- Java 8
- Hadoop 3.3.6
- Hadoop services running

## Commands Covered

- ls
- mkdir
- put
- cat
- cp
- mv
- du
- count
- get
- rm

## Result

Successfully executed basic HDFS commands and understood file operations in HDFS.
