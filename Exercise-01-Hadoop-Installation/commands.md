# Hadoop Installation Commands

## Check Java

```bash
java -version
```

## Check Hadoop

```bash
hadoop version
```

## Format NameNode

```bash
hdfs namenode -format
```

## Start Hadoop

```bash
start-dfs.sh
start-yarn.sh
```

## Check Processes

```bash
jps
```

## Check HDFS

```bash
hdfs dfsadmin -report
```
